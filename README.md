# Electronic-Document-Management:
**Веб-приложение, представляющее собой упрощенную систему электронного документооборота.**

## Стек технологий:
IntelliJ IDEA
Java SE
Apache Maven
Git
Spring Boot
Spring WebMVC
Spring Data JPA
Flyway
Postgresql
slf4j
Lombok
JUnit 5
Swagger

## Документация API
Swagger: http://localhost:8080/swagger-ui/index.html
